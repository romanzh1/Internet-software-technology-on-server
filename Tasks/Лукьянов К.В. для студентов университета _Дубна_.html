Лукьянов К.В. для студентов университета "Дубна"

3 курс, "Программные технологии интернет"

Задания

  1. Задание 1. Основы HTML. (3)

    1. Создать страницу-портфолио со следующей информацией: ФИО студента; номер группы; название и ссылка на страницу кафедры; название и ссылка на сервер ВУЗа.

    2. Дополнить страницу базовыми META-тегами.

    3. Разместить на странице изображение карты местности (города, района, страны), сделать ссылку с точки интереса (например, меторасположения ВУЗа) на соответствующий интернет-ресурс.

  2. Задание 2. Основы CSS. (3)

    1. Придумать визуальное оформление страницы из задания 1 (цвета, шрифты, фон итд) и реализовать его средствами CSS. СSS-правила должны подключаться из внешнего файла.

    2. Сделать изображение карты местности "резиновой" (т.е. занимающей всю ширину окна браузера). Связанная с изображением карта ссылок должна корректно указывать на соответствующие области интереса.

  3. Задание 3. Основы размещения элементов на странице. Формы. (3)

    1. Реализовать размещение двух блоков на странице рядом друг с другом. Ширина каждого блока должна составлять 50% от ширины окна браузера.

    2. Разместить в блоках форму "Запись на зачет/экзамен" со следующими полями ввода:

      • в левой колонке: ФИО студента; курс (выбор из списка); факультет (выбор из списка); номер группы
      • в правой колонке: предмет для сдачи (выбор из списка); преподаватель; день, месяц сдачи (выбор из списка); время; форма сдачи (очная/дистанционная, радио-переключатель)
      • под колонками по центру разместить кнопку "Записаться"

    3. Размещение блоков и оформление элементов в них осуществить с помощью CSS-правил. Не допускается использовать табличную верстку.

  4. Задание 4. Основы Javascript. Чтение/проверка данных из формы. (3)

    1. Дополните форму из задания 3 функционалом проверки заполненности всех полей. В случае, если поля не заполнены, кнопка "Записаться" должна выдавать ошибку. Реализовать средствами HTML5.

    2. Дополните форму из задания 3 функционалом чтения данных с помощью Javascript. Все прочитанные данные должны быть выведены ниже кнопки "Записаться" в виде таблички:

      ПРОВЕРЬТЕ И ПОДТВЕРДИТЕ ЗАПИСЬ НА ЗАЧЕТ
      ФИО:
      Курс:
      Факультет:
      Номер группы:
      Предмет:
      Преподаватель:
      Дата и время:
      Форма сдачи:

      Под этой таблицей необходимо разместить кнопку "Подтвердить"

  5. Задание 5. Изучение возможностей HTML5. Видео- и аудио- контент. (3)

    1. Создайте relax-страницу, на которой должен быть размещен видео- и аудио-контент стандартными средствами HTML5. Для этого:

      • Найдите любой видео-ролик с видами природы и сохраните его себе на компьютер. Найти подходящий по смыслу к видеоряду аудио-трек и сохраните его себе на компьютер.
      • Создайте HTML-страницу, к которой стандартными средствами HTML5 подключен видео-ролик. Ролик должен растягиваться на все окно браузера по ширине и высоте. Звук видео-ролика должен быть отключен. Проигрывание видео-ролика должно быть зациклено.

      • Поверх видео-ряда расположите аудио-плеер с загруженным аудио-треком. Попытайтесь сделать, чтобы проигрывание аудио начиналось автоматически вместе с загрузкой страницы.

  6. Задание 6. Изучение возможностей HTML5. Drag-n-drop. (3)

    1. Реализуйте функционал перетаскивания объектов между двумя контейнерами. Для этого создайте два контейнера, расположенных рядом друг с другом, имеющих одинаковый размер. В первом контейнере разместите 3 блока-прямоугольника. При клике на любой из этих блоков, удержании кнопки мыши и перетаскивании объекта в соседний контейнер блок должен туда переместится.

  7. Задание 7. Хранение данных в браузере. LocalStorage. (6)

    1. Реализуйте браузерную записную книжку со следующим механизмом работы:

      • На экране должны отображаться: поле для вывода всех сохраненных записей книжки (элемент формы textarea); поле для ввода новой записи (элемент формы input с типом text); кнопки "Сохранить" и "Стереть все записи"
      • При открытии страницы в поле для вывода всех записей должны отображаться все сохраненные в браузере записи
      • Когда пользователь набрал новую запись в соответствующем поле и нажал кнопку "Сохранить", запись должна быть сохранена в локальном хранилище браузера. После этого содержимое поля для вывода всех записей должно обновиться.
      • Когда пользователь нажал кнопку "Стереть все записи", данные из локального хранилище должны удалиться. После этого содержимое поля для вывода всех записей должно обновиться.

    Методические указания по выполнению задания

  8. Задание 8. Изучение возможностей HTML5. Canvas. (3)

    1. Нарисуйте на странице статичные графические примитивы (линии, круги, многоугольники) с помощью механизма Canvas.

    2. Добавьте одному из нарисованных графических примитивов анимацию, например, движение от левой стороны окна браузера к правой.

  9. Задание 9. Клиент-серверное взаимодействие. Основы SSI. Основы PHP. (6)

    1. Реализуйте счетчик посещаемости web-страницы. Для этого:

      • Установите и изучите принципы работы локального web-сервера (пакет Denwer, XAMPP, WAMP или какой-либо другой). Разместите на локальном web-сервере страницу, созданную в заданиях 1,2.
      • Создайте серверный скрипт "счетчик посещаемости" на языке PHP: при каждом запуске программы необходимо считать количестве посетителей из внешнего файла; увеличить число на 1; записывать результат в файл; выводить результат на экран.
      • Подключите разработанный скрипт к вашей странице с использованием SSI инструкций.

  10. Задание 10. Клиент-серверное взаимодействие. Работа с БД MySQL. (12)

    1. Разместите на локальном web-сервере форму "Запись на зачет/экзамен" (см.задания 3,4).

    2. Создайте с помощью инструмента phpmyadmin базу данных MySQL. Структура БД должна соответствовать полям формы.

    3. Напишите PHP-скрипт, который принимает данные формы, отправленные методом POST, и сохраняет их в БД. По результатам работы скрипт должен формировать ответ об успешном или ошибочном сохранении данных.

    4. Напишите PHP-скрипт для администратора, который показывает в виде таблицы список всех записавшихся на зачет/экзамен. Добавьте к каждой записи кнопку "Удалить", по нажатию на которую запись будет удаляться из БД.

  11. Задание 11. Клиент-серверное взаимодействие. Основы AJAX. (6)

    1. Разработайте web-приложение с поддержкой AutoSuggest на основе AJAX. Предполагается, что пользователь может вводить в текстовое поле формы название (например, автомобильной марки, рубрики, должности и т.п.), получая при этом динамически сформированный список вариантов названий, соответствующих уже введенным символам, без перезагрузки страницы.

    Особенности реализации: названия марок автомобилей должны хранится в БД

  12. Задание 12. Основы jQuery. Анимационные эффекты. (6)

    1. Разработайте веб-страницу и скрипт для фотогалереи с использованием фреймворка jQuery. Предполагается, что на странице выводится не меньше 12 уменьшенных эскизов фотографий (например, 3 ряда по 4 эскиза). При клике на фотографию поверх всех окон появляется модальное окно с полноразмерной фотографией. Также в этом окне должен быть крестик, реализующий функционал "закрыть фото", т.е. при клике на него большая фотография скрывается.

      Исходные данные: фреймворк jQuery, фотографии

  13. Задание 13. Основы WebScraping. (9)

    1. Разработайте php-скрипт, который отображает официальные курсы четырех валют (USD, EUR, JPY, GBP) на текущую дату по отношению к рублю, устанавливаемые ЦБ РФ. Данные с курсами валют необходимо подгружать методом web scraping с официального сайта ЦБ. Текущая дата определяется php-скриптом. Выборку данных из списка курсов осуществить с помощью регулярных выражений. Результаты должны быть представлены в следующем виде:

      100 Японских иен (JPY) = 58,4301 руб.
1 Фунт стерлингов Соединенного королевста (GBP) = 84,2522 руб.

      Исходные данные: cтраница с курсами валют на сайте ЦБ
